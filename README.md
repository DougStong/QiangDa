# 福建强大米业有限公司 — 官方网站

生态自然风格的单页企业官网，展示公司信息、产品与联系方式。

## 本地预览

在项目目录下任选一种方式：

```bash
# Python 3
python3 -m http.server 8080

# 或 Node.js（需安装 npx）
npx serve .
```

浏览器访问：<http://localhost:8080>

也可直接双击 `index.html` 用浏览器打开（部分字体需联网加载）。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 页面结构与内容 |
| `styles.css` | 样式与响应式布局 |
| `script.js` | 导航、滚动动画、留言表单 |

## 部署建议

- 上传 `index.html`、`styles.css`、`script.js` 至任意静态托管（阿里云 OSS、腾讯云 COS、GitHub Pages、Vercel 等）。
- 绑定企业域名并配置 HTTPS。
- 如需真实在线留言，可后续接入 Formspree、企业邮箱 API 或自建后端。
